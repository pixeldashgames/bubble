# Bubble

A 3D base-defense game built in **Godot 4.4** in 48 hours for **Global Game Jam 2025** (theme: *Bubble*) by the Pixel Dash Games team.

Set in the city of **Last Hope**, you place extractors to harvest resources, build turrets to fend off waves of enemies, and talk to the city's inhabitants along the way.

## Gameplay systems

- **Buildable structures** — resource extractors and defensive turrets, each with placement/management systems
- **Wave-based combat** — data-defined enemy waves, an enemy spawner, health and weak-point mechanics, laser beams
- **NPC dialog system** — a cast of characters (a scientist, a grandmother, a barista...) plus an AI-assistant voice guiding the player
- **Interaction system** — pickupable and long-press interactables with shader-based outline highlighting
- **Input handling** — built on the [GUIDE](https://github.com/godotneers/G.U.I.D.E) addon (mapping contexts, remapping support)

## Running the game

1. Open the project in **Godot 4.4+**
2. Run the main scene (`scenes/game_scene.tscn`)
